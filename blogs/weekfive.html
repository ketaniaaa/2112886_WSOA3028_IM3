Week Five: What is Metadata?

ketania govender 3 May 2021

Metadata, in summary, is data about other data. This may seem confusing but it is crucial for creating web pages. Metadata provides information about the web page that describes what the data contained is. This, at first can seen similar to semantic markup, in the sense that metadata allows for better information regarding the content of the page but unlike semantic markup, metadata does not structure the mark up. Metadata rathe provides a larger overview of the data that is provided.

Metadata is crucial for information systems and can be used in various manners because there is a variety of metadata. Chapple (2020) suggests five broad categories of metadata. These include descriptive metadata, rights, technical metadata, preservation metadata and markup languages. This indicates that semantic markup forms a part of metadata. Metadata can be manually created or automatically generated to provide rudimentary information about the page or document in question.

Metadata is often stored within the document in a manner that isn’t visible to the user necessarily – like the head of an HTML document- but can be accessed by the browser and search engine. Metadata directly impacts the success of a website. This is because search engines parse through the metadata of webpages for key words and content related to a user’s search in order to provide relevant webpages. This means that metadata drives the visibility of a website. In HTML, one can indicate the author, keywords and a brief description of the pages content which will be recognized by the browser. This can be seen in this webpage. The metadata on this page would include keywords such as metadata , HTML head , SEO and Search Engine Visibility along with information for the search engine and browser indicating that this is an English webpage.

The other uses of metadata can be used within tracking to better understand the analytics of a site and defining datasets. Metadata is crucial in creating a web environment that is relevant to todays users. We are used to being provided with information in an easy and quick manner. This efficiency of finding information is reliant on the appropriate use of metadata. This can be seen in the way social media is curated. I am sure that like me, you have been on a social media site such as pinterest and pinned a picture of a product you liked, just to be shown suggestions of similar products. This shows the use of metadata to suggest relevant information as the system has usually parsed through the metadata of files to show you similar content, but also demonstrates the use of metadata to track users. While tracking is still an ethical conundrum, I think that it is clear that metadata makes using the internet a whole lot easier and I would definitely prefer to metadata to be used to make my online experience efficient!
